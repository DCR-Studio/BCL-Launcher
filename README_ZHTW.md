### 創新的命令列形式，可自訂、開放且以用戶為中心的Minecraft啟動器

## 什麼是BCL Launcher？
BCL Launcher，全稱Bad Craft Launcher，是一個創新的命令列Minecraft啟動器，秉持以用戶為中心的理念。  
本專案使用Python編寫，我們也開源了使用Rust編寫的核心模組，詳見[OpenBCLCore專案](https://github.com/DCR-Studio/OpenBCLCore)。
