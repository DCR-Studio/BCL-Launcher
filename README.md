## BCL Launcher是什么？
BCL Launcher，全称Bad Craft Launcher，是一个创新的命令行Minecraft启动器，秉持以用户为中心的理念。  
BCL Launcher项目使用Python编写，我们也开源了使用Rust编写的内核，详见[OpenBCLCore项目](https://github.com/DCR-Studio/OpenBCLCore)。
